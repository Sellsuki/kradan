# kradan

> A Vue.js project

![logo](https://rawgit.com/Sellsuki/kradan/master/icon.png)

## usage

``` bash
sudo npm install -g kradan
kradan
```
![logo](https://rawgit.com/devilmustcry/kradan/master/usage-tutorial.gif)

## Build Setup

``` bash
# install dependencies
npm install

# run unit tests
npm run unit

# run all tests
npm test
```

## Dev

``` bash
npm run dev
```
